# Crossposter Privacy Policy

Effective date: June 2, 2026

Crossposter is a private local or self-hosted publishing tool used to prepare and publish posts to social accounts controlled by the operator.

## Data Stored

Crossposter may store:

- social API credentials configured by the operator
- draft text and post titles
- uploaded media used for publishing
- local publish history

These values are stored in the operator's own local or self-hosted environment. The default local config file is `poster.config.local.json`, which is gitignored.

## Data Use

Crossposter uses configured credentials only to publish content the operator submits manually. Uploaded media and draft content are used only for publishing and local history.

## Third Parties

Crossposter sends content only to the social platforms selected by the operator during publishing. It does not sell personal information, run advertising pixels, or use tracking cookies.

## Security

The operator is responsible for keeping API keys, access tokens, refresh tokens, and local configuration private. Do not commit secrets to source control.
